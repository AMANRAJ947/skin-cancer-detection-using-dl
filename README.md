🧠 Skin Cancer Detection using Deep Learning

This project focuses on detecting skin cancer using deep learning techniques. It leverages Convolutional Neural Networks (CNNs) to classify skin lesion images as cancerous or non-cancerous, helping in early diagnosis and assisting medical professionals.

Early detection of skin cancer is crucial, as it significantly increases the chances of successful treatment and survival.

📌 Project Overview

Skin cancer is one of the most common types of cancer worldwide. Traditional diagnosis relies on dermatologists and biopsy procedures, but deep learning models can assist by providing fast and accurate predictions from images.

This project:

Uses image classification techniques
Trains a deep learning model on skin lesion datasets
Predicts whether a lesion is benign or malignant

⚙️ Installation
Clone the repository:
git clone https://github.com/AMANRAJ947/skin-cancer-detection-using-dl.git
cd skin-cancer-detection-using-dl
Create virtual environment (optional but recommended):
python -m venv venv
venv\Scripts\activate   # Windows
Install dependencies:
pip install -r requirements.txt
▶️ Usage

Run the main file:

python main.py

For Jupyter Notebook:

jupyter notebook
